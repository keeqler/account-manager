# Account Manager

An account managing web app built entirely with NodeJS, for learning purposes only.

# Built with

### Back-end:

- Express
- Sequelize (Postgres)
- Redis
- Bee Queue
- JSON Web Token

### Front-end:

- React
- Redux
- Redux Saga
- Styled Components
- Unform (forms)
